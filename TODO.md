# PROGETTO: Sito Web Consulenza Anti-Esternalizzazione

## TECH STACK
- React 18+ con Vite
- Tailwind CSS 3.x
- Framer Motion (animazioni)
- React Router DOM
- Lucide React (icone)

## STRUTTURA COMPONENTI

### 1. Layout Base
- [ ] Navbar responsive con menu mobile hamburger
- [ ] Footer con link social, contatti, CTA finale
- [ ] Layout wrapper con smooth scroll

### 2. Sezione Hero
- [ ] Headline d'impatto: "Le aziende bruciano milioni in outsourcing. Io li salvo."
- [ ] Sottotitolo con value proposition
- [ ] CTA primario: "Prenota Consulenza Gratuita"
- [ ] CTA secondario: "Scopri quanto stai perdendo"
- [ ] Animazione di sfondo (gradient mesh o particelle sottili)

### 3. Sezione "Il Problema"
- [ ] Statistiche animate sui costi outsourcing PA/aziende
- [ ] Cards con i 3-4 problemi principali dell'esternalizzazione
- [ ] Grafico/infografica costi nascosti
- [ ] Tone: professionale ma incisivo

### 4. Sezione "La Soluzione"
- [ ] Spiegazione AI Agents + Vibe Coding (accessibile ai non-tecnici)
- [ ] Confronto visivo: Prima vs Dopo
- [ ] Timeline del processo di transizione
- [ ] Badge "Nessun rischio - Paghi solo sui risultati"

### 5. Sezione "Chi Sono"
- [ ] Foto professionale
- [ ] Timeline carriera interattiva
- [ ] Competenze chiave con progress bars o tags
- [ ] Quote personale / mission statement
- [ ] Link LinkedIn e altri profili

### 6. Sezione Pricing
- [ ] Card "Primo Consulto": GRATIS
- [ ] Card "Modello Success Fee": 50% del risparmio generato
- [ ] Simulatore interattivo: "Quanto spendi ora?" → "Quanto risparmieresti"
- [ ] FAQ accordion sul modello di pagamento
- [ ] Trust badges (garanzia, trasparenza, etc.)

### 7. Sezione Pubblica Amministrazione
- [ ] Focus specifico sui danni al pubblico interesse
- [ ] Riferimenti normativi (se applicabili)
- [ ] Casi studio anonimi o generici
- [ ] CTA dedicato per enti pubblici

### 8. Contatti / CTA Finale
- [ ] Form di contatto con validazione
- [ ] Integrazione Calendly o simile per booking
- [ ] Mappa o indicazione area di servizio
- [ ] Social proof: testimonial o loghi clienti

## DESIGN SYSTEM
- [ ] Palette colori: professionale (blu/grigio) + accento energia (verde o arancio)
- [ ] Typography: Inter o similar (moderna, leggibile)
- [ ] Spacing consistente (8px grid)
- [ ] Dark mode toggle (opzionale ma consigliato)
- [ ] Micro-animazioni hover su cards e bottoni

## OTTIMIZZAZIONI
- [ ] SEO meta tags per ogni pagina
- [ ] Open Graph per social sharing
- [ ] Performance: lazy loading immagini
- [ ] Accessibilità: ARIA labels, contrasto colori
- [ ] Mobile-first responsive design

- [ ] Bio professionale estesa

- [ ] Descrizione processo di lavoro
